# 🛒 freshsaver: Hyperlocal Flash Sale Platform

**Vashisht Hackathon 3.0 Submission** ## 🚨 The Problem
Local retailers often face challenges in clearing overstocked or near-expiry inventory due to limited digital presence. Simultaneously, nearby customers miss out on these time-sensitive deals, and the disconnect results in lost revenue and increased food/product waste. There is no real-time mechanism connecting supply and demand at a hyperlocal level.

## 💡 Our Solution: DealDrop
DealDrop is a real-time, hyperlocal marketplace connecting local businesses with nearby consumers. 
* **For Retailers:** A unified dashboard to list perishable goods, set expiry times, and receive intelligent, time-based discount recommendations to clear stock before it expires.
* **For Consumers:** A high-energy mobile-first feed showing nearby flash deals, complete with expiry countdown timers and instant map navigation to the store. 

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite, Tailwind CSS, Lucide Icons
* **Deployment:** Vercel 
* **Architecture:** Responsive Web App (Desktop-optimized for Retailers, Mobile-optimized for Consumers)

## 🚀 Setup Instructions (Local Development)
To run this project locally on your machine:

1. Clone the repository:
   `git clone <YOUR_GITHUB_REPO_URL>`
2. Navigate into the project directory:
   `cd <PROJECT_FOLDER_NAME>`
3. Install the dependencies:
   `npm install`
4. Start the development server:
   `npm run dev`
5. Open your browser and visit `http://localhost:5173`
